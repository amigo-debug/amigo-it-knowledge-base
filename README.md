# Amigo IT 知識庫

恩海居資訊中心的 IT 術語知識庫。用廣東話口語解釋技術概念，配合院舍場景比喻，讓前線員工輕鬆理解 IT 術語。

## 分類

- 🔐 **登入與安全** — 登入、權限、加密、身份驗證
- 🗄️ **資料庫** — 數據儲存、查詢、結構設計
- 🖥️ **前端介面** — UI、網頁、App、畫面設計
- 🏗️ **系統架構** — 系統設計、API、部署策略
- 📱 **硬件與連線** — NFC、QR Code、網絡、設備
- ⚙️ **自動化** — 自動化流程、排程、觸發器
- 📄 **文件處理** — PDF、Excel、Word、OCR
- 🌐 **網絡** — Wi-Fi、域名、SSL、協議

## 使用方式

1. 瀏覽 [線上知識庫](https://amigo-debug.github.io/amigo-it-knowledge-base/)
2. 按分類查找術語
3. 每個術語都有廣東話解釋和院舍比喻

## 更新流程

新術語會自動從 `amigo-it-knowledge-base` 專案同步到此倉庫。
